32 /* To add a new kind of warning, add an
33  * if(_curl_is_sometype_option(_curl_opt))
34  * if(!_curl_is_sometype(value))
35  * _curl_easy_setopt_err_sometype();
36  * block and define _curl_is_sometype_option, _curl_is_sometype and
37  * _curl_easy_setopt_err_sometype below
38  *
39  * NOTE: We use two nested 'if' statements here instead of the && operator, in
40  * order to work around gcc bug #32061. It affects only gcc 4.3.x/4.4.x
41  * when compiling with -Wlogical-op.
42  *
43  * To add an option that uses the same type as an existing option, you'll just
44  * need to extend the appropriate _curl_*_option macro
45  */
